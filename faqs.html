How can I install SSL?

After you have bought the SSL certificate, go to ./app/config/app.php file, find 'url', replace your site url to https version.

How can I redirect my website to the SSL version?

Make sure you have SSL working on your domain, and 'url' is set to SSL version in ./app/config/app.php file.
Open .htaccess file, under "RewriteEngine On", copy and the paste the code below:

RewriteCond %{HTTPS} off RewriteRule (.*) https://%{HTTP_HOST}%{REQUEST_URI} [R,L]
